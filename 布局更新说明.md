# 页面布局更新说明

## 更新内容

本次更新对页面导航栏进行了重新设计，采用三栏式布局：

### 1. 布局结构

```
┌───────────────────────────────────────────────────────────────────────┐
│              瓯越交建洞头厂区智慧梁场信息化管控平台                  │
├──────────────┬────────────────────────┬───────────────────────────────┤
│              │                        │                               │
│ 项目概况     │     项目概况           │    设备管理                   │
│ 生产信息     │     瓯越交建洞头厂区   │    材料管理                   │
│ 梁场管理     │     智慧梁场           │    安全管理                   │
│ 质量管理     │                        │    系统设置                   │
│              │                        │                               │
└──────────────┴────────────────────────┴───────────────────────────────┘
    左侧4按钮         中间标题区域            右侧4按钮
```

### 2. 布局特点

**左侧按钮组**：
- 4个导航按钮（项目概况、生产信息、梁场管理、质量管理）
- 垂直排列（可根据需要改为水平）
- 玻璃拟态效果

**中间标题区域**：
- 显示当前选中功能名称（大字）
- 显示副标题（瓯越交建洞头厂区智慧梁场）
- 居中固定
- 实时更新

**右侧按钮组**：
- 4个导航按钮（设备管理、材料管理、安全管理、系统设置）
- 垂直排列（可根据需要改为水平）
- 玻璃拟态效果

### 3. 样式升级

**按钮样式（玻璃拟态）**：
```css
- 半透明背景: rgba(30, 41, 59, 0.4)
- 模糊效果: backdrop-filter: blur(5px)
- 渐变边框: rgba(56, 189, 248, 0.4)
- 阴影: 0 4px 6px rgba(2, 8, 20, 0.3)
- 内发光: inset 0 1px 1px rgba(255, 255, 255, 0.1)
```

**悬停效果**：
- 上浮: transform: translateY(-3px)
- 边框发光: border-color: rgba(56, 189, 248, 0.8)
- 阴影增强: 0 6px 15px rgba(56, 189, 248, 0.3)

**激活状态**：
- 渐变背景: linear-gradient(135deg, #3b82f6 0%, #6366f1 100%)
- 脉冲动画: 顶部蓝色线条闪烁
- 内发光: 白色渐变叠加

### 4. 交互功能

**按钮点击**：
1. 移除所有按钮的active类
2. 为当前按钮添加active类
3. 更新中间标题文本
4. 发送UE消息: `OnNavButtonClicked`
5. 记录日志

**中间标题更新**：
```javascript
updateNavTitle(section) {
    const titleElement = document.querySelector('.nav-title-text');
    const sectionName = this.getSectionName(section);
    titleElement.textContent = sectionName;
}
```

## 自定义配置

### 修改按钮文本

编辑 `src/pages/test.html`：

```html
<!-- 左侧按钮组 -->
<div class="nav-group left-group">
    <button class="nav-button" data-section="your-section">你的文本</button>
</div>

<!-- 右侧按钮组 -->
<div class="nav-group right-group">
    <button class="nav-button" data-section="your-section">你的文本</button>
</div>
```

### 修改样式

编辑 `src/pages/test.html` 的 `<style>` 部分：

```css
/* 按钮背景 */
.nav-button {
    background: rgba(30, 41, 59, 0.4);
    border: 1px solid rgba(56, 189, 248, 0.4);
}

/* 激活状态 */
.nav-button.active {
    background: linear-gradient(135deg, #3b82f6 0%, #6366f1 100%);
}

/* 中间标题 */
.nav-title-text {
    font-size: 1.75rem;
    background: linear-gradient(90deg, #38bdf8 0%, #818cf8 100%);
}
```

### 修改布局方向

如果要改为水平排列：

```css
.nav-group {
    display: flex;
    flex-direction: row;  /* 改为水平 */
    gap: 1rem;
}
```
