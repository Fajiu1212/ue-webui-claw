# 预览页面说明

## 页面功能

### 1. 顶部科技导航栏
- **背景**: 深蓝色科技风格
- **边框**: 蓝色科技感边框
- **发光线条**: 底部蓝色发光线条（带脉冲动画）
- **位置**: 页面顶部固定
- **高度**: 80px

### 2. 导航栏一行布局
- **左侧**: 3个按钮（项目概况、生产信息、安全信息）
- **中间**: 大标题（瓯越交建洞头厂区智慧梁厂信息化管控平台）
- **右侧**: 3个按钮（质量信息、物资信息、AI工序识别）

### 3. 导航按钮（6个）

**左侧**:
1. ✅ 项目概况（默认激活）
2. 生产信息
3. 安全信息

**右侧**:
4. 质量信息
5. 物资信息
6. AI工序识别

### 4. 按钮功能
- 点击任意按钮会激活该按钮（高亮显示）
- 中间标题会更新为当前选中的功能名称
- 所有按钮在同一水平行内
- 发送UE消息：`OnNavButtonClicked`
- 消息数据包含：
  - `section`: 区域标识（如"project-overview"）
  - `sectionName`: 区域显示名称（如"项目概况"）
  - `timestamp`: 时间戳

### 5. 样式特点

**导航栏背景**:
- 深蓝色渐变（#0a0e27 → #0f172a）
- 科技蓝边框
- 底部蓝色发光线条（带脉冲动画）

**按钮样式**:
- 半透明背景
- 蓝色边框
- 悬停发光效果
- 激活状态（蓝到紫渐变）
- 光泽流动动画

**中间标题**:
- 大字显示
- 蓝色发光文字
- 阴影效果

## 预览方法

### 方法1：直接双击打开
直接双击 `src/pages/test.html` 文件

### 方法2：使用HTTP服务器

#### 方式A：使用Node.js服务器
```bash
# 在项目根目录执行
cd g:\Claw_Project\Claw_WebUI
node start-server.js
```
服务器会自动在浏览器中打开页面

#### 方式B：使用Python服务器（如果有Python）
```bash
cd g:\Claw_Project\Claw_WebUI\src\pages
python -m http.server 8080
# 访问 http://localhost:8080/test.html
```

#### 方式C：使用Vite开发服务器
```bash
cd g:\Claw_Project\Claw_WebUI
npm install
npm run dev
# 访问 http://localhost:5173/pages/test.html
```

### 方法3：使用VS Code插件
在VS Code中安装 "Live Server" 插件，右键点击test.html选择 "Open with Live Server"

## 与UE集成测试

1. 在UE中创建Widget Blueprint

3. 配置URL指向test.html文件

