## 🔧 防火墙设置

### 快速添加防火墙规则

以管理员身份运行PowerShell：
```powershell
New-NetFirewallRule -DisplayName "UE5.5 WebUI" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow
```

### 验证端口

```bash
netstat -an | findstr "8080"
```

应该看到：
```
TCP    0.0.0.0:8080           0.0.0.0:0              LISTENING
```

## 🔍 故障排查

### 同事无法访问？

1. **检查防火墙**: 确保端口8080已开放
2. **检查网络**: 确保在同一局域网
3. **检查IP地址**: 确认IP地址是否正确

### 页面显示异常？

1. **清除缓存**: Ctrl + F5 强制刷新
2. **检查浏览器**: 使用现代浏览器（Chrome/Firefox/Edge）
3. **查看日志**: 打开浏览器控制台查看错误

### UE通信失败？

1. **检查插件**: WebUI插件已启用
2. **检查蓝图**: OnInterfaceEvent已绑定
3. **检查日志**: 查看UE输出日志
